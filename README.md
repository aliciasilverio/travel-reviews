# Travel Review App

Post about the places you’ve travelled to and enjoyed as well as the places to avoid.

## Technologies: 
HTML/CSS

Javascript

React

Node.js / Express

MongoDB


## MVP:
- A working full-stack application, using the MERN stack
- Be deployed online and accessible to the public via Heroku
